## 实现计划

### 1. 项目依赖安装
- 安装最新版的 shadcn/ui 及相关依赖
- 安装最新版的 zustand
- 安装 Tauri 快捷键相关的插件

### 2. 项目结构调整
- 创建 src/components 目录存放组件
- 创建 src/store 目录存放状态管理
- 创建 src/hooks 目录存放自定义 hooks
- 创建 src/types 目录存放类型定义

### 3. 核心功能实现

#### 3.1 快捷键系统
- 创建 `src/store/shortcut.ts` 使用 zustand 管理快捷键配置
- 创建 `src/hooks/useShortcut.ts` 自定义 hook 处理快捷键监听
- 实现灵活的快捷键注册机制，支持后续扩展

#### 3.2 主窗口组件
- 重构 `App.tsx`，实现以输入框为核心的主界面
- 使用 shadcn/ui 的 Input 组件
- 实现窗口的显示/隐藏逻辑

#### 3.3 设置页面
- 创建 `src/components/Settings.tsx` 设置组件
- 实现快捷键配置界面，允许用户自定义快捷键
- 使用 shadcn/ui 的 Form、Input 等组件

#### 3.4 Tauri 配置调整
- 更新 `src-tauri/tauri.conf.json`，配置窗口属性（如透明背景、无边框等）
- 添加快捷键相关的 Tauri 命令

### 4. 样式与布局
- 使用 shadcn/ui 的样式系统
- 实现简洁现代的 UI 设计
- 确保主窗口输入框居中显示

### 5. 测试与优化
- 测试快捷键功能是否正常工作
- 测试设置页面的配置是否能正确保存
- 优化窗口显示/隐藏的动画效果

### 6. 代码检查
- 确保代码符合 TypeScript 规范
- 检查是否有语法错误或警告

## 技术栈
- React 19
- TypeScript 5
- Tauri 2
- shadcn/ui (最新版)
- zustand (最新版)

## 预期效果
- 应用启动后默认隐藏，可通过快捷键呼出
- 呼出后显示一个居中的输入框
- 用户可在设置中自定义呼出快捷键
- 快捷键系统设计灵活，支持后续扩展其他功能
